# Linux TomCat 
 
Установка и настрока Apache Tomcat под Linux 

```
apt install tomcat8
```
 Пакет подтягивает все зависимости и устанавливает init скрипты 

Устанавливаем документацию и Tomcat_admin
Наслаждаемся =)
